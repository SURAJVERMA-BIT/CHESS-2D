# Changelog

All notable changes to Grandmaster Chess Suite will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2024-01-XX

### Added
- Full castling implementation (kingside and queenside)
- En passant capture functionality
- Pawn promotion with piece selection modal
- Stalemate detection
- 50-move rule enforcement
- Move history with undo/redo functionality
- PGN export feature
- Game timers (5, 10, 15, 30 minutes, unlimited)
- Move hints system
- Evaluation bar visualization
- Game statistics (moves, captures)
- Proper algebraic notation
- Coordinate labels (files a-h, ranks 1-8)
- AI opponent with 5 difficulty levels
- Minimax algorithm with alpha-beta pruning
- Clickable move history navigation
- Premium glassmorphism UI design
- Enhanced sidebar with controls
- Timer warnings for low time
- Multiple premium themes (Onyx, Ivory, Sapphire, Ruby)
- Settings modal with comprehensive options
- Responsive design improvements

### Changed
- Improved move validation logic
- Enhanced visual feedback for moves
- Better check/checkmate detection
- Refined sound effects
- Updated UI/UX for premium feel
- Improved code organization and structure

### Fixed
- Fixed piece movement validation
- Corrected king position tracking
- Improved castling rights management
- Fixed en passant target tracking
- Better error handling

## [1.0.0] - 2024-01-XX

### Added
- Initial release
- Basic chess game functionality
- Move validation
- Check detection
- Basic UI
- Sound effects
- Move assist mode
- Multiple themes
